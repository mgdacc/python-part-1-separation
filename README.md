4. [Loops](https://mgdacc.github.io/python-loops/)
5. [Type Casting](https://mgdacc.github.io/python-type-casting/)
6. [Exceptions](https://mgdacc.github.io/python-exceptions/)
7. [Functions](https://mgdacc.github.io/python-functions/)
8. [Lists](https://mgdacc.github.io/python-lists/)
9. [Tuples](https://mgdacc.github.io/python-tuples/)
10. [Sets](https://mgdacc.github.io/python-sets/)
